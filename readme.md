## 🌍 About the Project

**Zero Hunger** is a responsive web application designed to combat food insecurity. By connecting donors with organizations and individuals in need, the platform promotes a culture of giving and social responsibility. The interface is meticulously crafted to be engaging, professional, and accessible.

## ✨ Features

- **Modern User Interface:** Built with a clean, glassmorphic design and rich aesthetics.
- **Donation Portal:** A seamless, user-friendly form for pledging food donations (quantities, photos, and messages).
- **Availability Dashboard:** A real-time view of recent donations and available food resources.
- **Responsive Layout:** Fully optimized for mobile, tablet, and desktop viewports.
- **Interactive Elements:** Smooth fade-in transitions and micro-animations for an enhanced user experience.

## 🛠 Tech Stack

- **HTML5:** Semantic and accessible markup.
- **Tailwind CSS:** Utility-first styling for rapid, premium UI development.
- **Vanilla JavaScript:** Lightweight DOM manipulation and form validation.

## 💻 Local Setup Instructions

If you'd like to run or modify this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mohdadnanmohi/zero_hunger.git
   cd zero_hunger
   ```

2. **Install dependencies:**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Build or watch the CSS:**
   ```bash
   # Build the tailwind output once
   npm run build

   # Or, watch for changes during development
   npm run watch
   ```

4. **Run the application:**
   Open the `index.html` file in your preferred browser, or use a tool like Live Server.
